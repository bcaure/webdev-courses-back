GAMES API

Route /games (method=GET)

Returns all games: 

      [
        {
          "id": number,
          "name": string,
          "summary": string,
          "releaseDate": timestamp,
          "photo": string,
          "similarGames": comma separated game ids
        }
      ]

Route /gamesreview (method=GET)

Returns all games reviews: 

      [
        {
          "id": number,
          "comment": string,
          "note": number (from 1 to 10),
          "game_id": number
        }
      ]

Route /gamesreview (method=POST)

Creates a review for a given game Mandatory fields: "comment", "note", "gameId"

  • Error: return text message with error description
  • Success: return text message "X ROW(S) INSERTED"

Route /gamesreview/ (method=DELETE)

Deletes a game review

  • Error: return text message with error description
  • Success: return text message "X ROW(S) DELETED"
